# 算法说明
## 数据类型
### 菜品

|     数据     | 内容 |    备注  |
| ------------ | --- | ---------------|
| 名称 |  菜品名称 |  |
| 主料 |  必要的食材 | 最多5种，最少1种 |
| 配料 |  其他不必要的食材 | 最多20种，最少0种 |
| 时间 |  所需时间 | 单位统一为分钟 |
| 选项 |  已选／未选 | 用户选择 |

例子：
- 名称：西红柿炒鸡蛋
- 主料：番茄；鸡蛋
- 配料：葱花；姜；番茄酱
- 时间：5分钟
